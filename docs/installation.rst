Installation
============

Editable install (recommended for development)
---------------------------------------------

.. code-block:: bash

   pip install -e ".[dev]"

Optional extras
---------------

Documentation tooling:

.. code-block:: bash

   pip install -e ".[docs]"

JAX optimization ecosystem (used in some advanced examples):

.. code-block:: bash

   pip install -e ".[opt]"

Plotting (for publication-style figures in some examples):

.. code-block:: bash

   pip install -e ".[viz]"
