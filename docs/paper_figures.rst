Paper figures (reproduced)
==========================

This page includes low-resolution reproductions of key figures from the original SFINCS v3
paper, generated by running the corresponding sfincs_jax cases. The goal is to provide
quick visual targets for parity and regression checks, not publication-quality images.

Generate the figures with

.. code-block:: bash

   python examples/publication_figures/generate_sfincs_paper_figs.py --fast

Omit ``--fast`` to use the default scan resolution.
